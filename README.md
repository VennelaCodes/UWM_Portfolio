## Projects

### Project 1: Operational Data Review & KPI Analysis
Analyzes 52,000+ operational records to validate data quality, calculate KPIs against targets, and generate executive-ready summaries.

**Key Results:**
- 95.1% data quality score across 52,000 records
- 6 automated validation rules catching 2,544 errors
- KPI vs target tracking with trend analysis and recommendations

**Tech:** Python, SQL, Excel

```bash
cd Project1_KPI_Analysis
python3 kpi_analysis.py
```

---

### Project 2: Data Governance & Reporting Optimization
Consolidates 3 separate data sources into standardized dashboards, designs 5 new composite metrics, and automates recurring reporting workflows.

**Key Results:**
- 25%+ reduction in reporting preparation time
- 5 new composite metrics (Operational Efficiency Index, Client Experience Score, etc.)
- 17 data discrepancies identified through automated root cause analysis

**Tech:** Python, SQL Server, Excel

```bash
cd Project2_Data_Governance
python3 reporting_optimization.py
```

### Project 3: Ad-hoc Reporting & Process Efficiency
Handles 200 ad-hoc reporting requests across 5 business units with structured daily/weekly/monthly reports and pre-submission validation.

**Key Results:**

- Daily, weekly, and monthly reporting at 3 frequencies

## How to Run

All projects use Python standard library only — no additional packages required.

```bash
# Clone the repo
git clone https://github.com/VennelaCodes/UWM_Portfolio.git
cd UWM_Portfolio

# Run any project
python3 Project1_KPI_Analysis/kpi_analysis.py
python3 Project2_Data_Governance/reporting_optimization.py
python3 Project3_Adhoc_Reporting/scripts/adhoc_reporting.py
```

Open `index.html` in a browser to view the interactive portfolio website.
